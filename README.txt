From PHP folder
- Put the folder "fuzzy-robot" in your Apache server. It should be accesed via https://xxxx/fuzzy-robot, ex. https://localhost/fuzzy-robot

From SQL Folder
- First create the arcade_game table in phpmyadmin
- Import the SQL file "arcade_game", Then select "arcade_game" database.
- Import "arcade_game_previleges" while "arcade_game" database is selected.
